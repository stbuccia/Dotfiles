# Dotfiles

my dotfiles ;)

## Dependences

+ curl
+ vim
+ vim-python-jedi
+ fd-find
+ i3
+ i3-lock-fancy
+ rofi
+ polybar
+ playerctl
+ thefuck

## Installazione

```
git clone https://github.com/stbuccia/Dotfiles.git ~/Dotfiles
cd !$
./install.sh
```

Per installare i Plugin in vim digita all'interno di vim
PlugInstall
